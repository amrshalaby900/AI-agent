# 🤝 دليل المساهمة في Amr AI

شكراً لاهتمامك بالمساهمة في مشروع Amr AI! نحن نرحب بجميع المساهمات.

## 📋 قبل البدء

- اقرأ [README.md](README.md) لفهم المشروع
- تأكد من أن لديك Python 3.11+ و Node.js 18+
- استنسخ المستودع وأنشئ فرع جديد

## 🔄 خطوات المساهمة

### 1. Fork المستودع
```bash
# انقر على زر Fork على GitHub
```

### 2. استنسخ نسختك
```bash
git clone https://github.com/YOUR_USERNAME/AI-agent.git
cd AI-agent
```

### 3. أنشئ فرع جديد
```bash
git checkout -b feature/your-feature-name
```

### 4. قم بالتغييرات
- اكتب الكود بشكل نظيف وواضح
- أضف تعليقات حيث يكون ضروري
- اتبع معايير الكود الموجودة

### 5. اختبر التغييرات
```bash
# للـ Backend
python -m pytest

# للـ Frontend
cd frontend && pnpm test
```

### 6. Commit التغييرات
```bash
git add .
git commit -m "Add: وصف واضح للتغييرات"
```

### 7. Push إلى الفرع
```bash
git push origin feature/your-feature-name
```

### 8. فتح Pull Request
- اذهب إلى GitHub
- اضغط على "New Pull Request"
- اشرح التغييرات التي قمت بها

## 📝 معايير الكود

### Python
- استخدم PEP 8
- أضف docstrings للدوال
- استخدم type hints

### JavaScript/React
- استخدم ESLint
- اكتب مكونات وظيفية
- استخدم Hooks

## 🐛 الإبلاغ عن الأخطاء

إذا وجدت خطأ:

1. تحقق من أنه لم يتم الإبلاغ عنه مسبقاً
2. فتح Issue جديد
3. اشرح الخطأ بوضوح
4. أضف خطوات للتكرار

## 💡 اقتراح ميزات جديدة

1. فتح Issue جديد
2. اشرح الميزة المطلوبة
3. اشرح الفائدة
4. انتظر التعليقات

## 📚 معايير الجودة

- ✅ الكود يجب أن يمر جميع الاختبارات
- ✅ لا توجد تحذيرات من linter
- ✅ التوثيق محدث
- ✅ الكود نظيف وسهل الفهم
